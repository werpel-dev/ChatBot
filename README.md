# 📚 Chat Estudo - Assistente de Aprendizado

O **Chat Estudo** é uma aplicação desktop desenvolvida em **Python** utilizando a biblioteca **Tkinter** para a construção da interface gráfica. O projeto foi criado com o objetivo de aplicar conceitos fundamentais de programação, lógica computacional e desenvolvimento de interfaces, simulando um chatbot capaz de interagir com o usuário e sugerir conteúdos para estudo.

Embora seja um projeto de aprendizagem, ele demonstra a utilização de estruturas condicionais, manipulação de eventos, organização de código e construção de aplicações gráficas utilizando recursos nativos do Python.

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para colocar em prática conhecimentos adquiridos durante os estudos em programação, proporcionando experiência com:

* Desenvolvimento de aplicações desktop em Python;
* Construção de interfaces gráficas utilizando Tkinter;
* Lógica de programação e tomada de decisões;

* Manipulação de eventos e interação com o usuário;
* Boas práticas de desenvolvimento de software.

---

## 🚀 Funcionalidades

O Chat Estudo oferece uma interação simples e intuitiva entre o usuário e a aplicação.

Atualmente, o sistema possui as seguintes funcionalidades:

* Interface gráfica em formato de chat;
* Conversação simples com respostas automáticas;
* Sugestão aleatória de matérias para estudo;
* Organização da rotina de estudos de forma interativa;
* Atualização das mensagens em tempo real;
* Layout leve e de fácil utilização.

## ▶️ Como Executar

### Pré-requisitos

* Python 3.10 ou superior instalado.

### Passos

1. Faça o download ou clone este repositório.
2. Abra o projeto em uma IDE de sua preferência (VS Code, PyCharm ou IDLE).
3. Execute o arquivo principal do projeto.

Após iniciar a aplicação:

* Digite **"oi"** para iniciar a conversa.
* Em seguida, digite **"qual matéria?"** para receber uma sugestão de disciplina para estudar.
* Continue interagindo com o chatbot por meio das mensagens disponíveis.
